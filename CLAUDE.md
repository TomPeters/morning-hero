# Morning Hero — Claude Guide

## Project Overview

Morning Hero is a kids' app to help them complete their morning jobs when getting ready for school. The target users are children (primary audience) with parents as secondary users/admins.

## Key Goals

- Make morning routines fun and manageable for kids
- Simple, clear UI appropriate for young children
- Allow parents to configure the list of jobs and order
